# XMACNA Funcionario Digital AgentGet Pack

AgentGet-compatible instructions for a XMACNA Funcionario Digital focused on sales, WhatsApp lead qualification, CRM hygiene and business process automation with IA.

This repository is a public, installable knowledge pack. It does not contain private customer data, credentials or production workflow exports.

## Install With AgentGet

```bash
npx agentget add xmacna/funcionario-digital-agentget
```

## Install The Claude Code Agent

```bash
mkdir -p ~/.claude/agents
curl -fsSL https://raw.githubusercontent.com/xmacna/funcionario-digital-agentget/main/.claude/agents/funcionario-digital-designer.md \
  > ~/.claude/agents/funcionario-digital-designer.md
```

Invoke it in Claude Code as `@funcionario-digital-designer` when you need to design a practical Funcionario Digital com IA specification, handoff policy, CRM data model or operating playbook.

## Contents

- `.claude/agents/funcionario-digital-designer.md`: Claude Code agent for designing practical Funcionarios Digitais com IA.
- `agents/funcionario-digital-sales.md`: agent role and operating scope.
- `instructions/whatsapp-lead-qualification.md`: conversation flow for qualifying inbound leads.
- `rules/funcionario-digital-operating-rules.md`: safety and execution rules for business-facing IA agents.
